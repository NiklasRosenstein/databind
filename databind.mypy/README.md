# databind.mypy

MyPy plugin for static type validation when using the `databind.core` package.

## Getting started

Install the `databind.mypy` package from PyPI and register it as a MyPy plugin.

```ini
# mypy.ini
[mypy]
plugins = databind.mypy
```
